# 谷歌搜索语法

- "" 限定关键词
- intitle 限定标题
- allintitle 标题多个关键词
- intext 内容关键词
- inurl 限定网址关键词
- site 限定网址来源
- filetype 限定文件格式
  eg： filetype：pdf 母婴报告
